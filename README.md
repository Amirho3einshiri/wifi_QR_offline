## ✳ ویژگی‌ها
- عملکرد آفلاین — نیازی به اینترنت ندارد  
- پشتیبانی از انواع QR و بارکد  
- خروجی متنی مستقیم  
- سبک و قابل اجرا روی هر سیستم‌عاملی
